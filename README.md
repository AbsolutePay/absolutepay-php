# absolutepay/sdk (PHP)

Official PHP SDK for [AbsolutePay](https://absolutepay.io) — accept crypto payments, send
payouts, and reconcile. Early placeholder that reserves the Composer vendor; full client on the way.

- **Docs:** https://docs.absolutepay.io
- **API base:** `https://api.absolutepay.io`

## Install

```bash
composer require absolutepay/sdk
```
